About Me

Hello! I'm Franco Zarate, a passionate web developer based in Córdoba, Argentina. I specialize in creating modern, functional web experiences that combine attractive design with clean, efficient code.

My journey in web development began with a curiosity to understand how the digital technologies we use every day work. Since then, I've been constantly learning and honing my skills in HTML, CSS, JavaScript, and modern frameworks.

I focus on creating responsive websites that not only look great but also deliver an exceptional user experience. I strongly believe in writing clean, maintainable, and accessible code.

When I'm not programming, I enjoy exploring new technologies, contributing to open source projects, and sharing knowledge with the developer community.
